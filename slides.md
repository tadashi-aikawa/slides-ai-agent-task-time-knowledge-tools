# cmuxの推しポイント

- Verticalタブ
  - ワークスペースを縦に並べられる
  - 通知バッチがつく
  - cmuxコマンドでタイトルやステータスなどを切り替え可能
- cmuxコマンドで色々なことができる
  - cmux外からも操作可能
  - エージェントのオーケストレーションなども可能
- その他
  - ベースは[Ghostty](https://ghostty.org/)([libghostty](https://github.com/ghostty-org/ghostty?tab=readme-ov-file#cross-platform-libghostty-for-embeddable-terminals)を使用)
  - remote SSH や ブラウザなどにも対応

# カレントディレクトリ配下で特定文字列を含むパスに移動

```bash [~/.zshrc]
iz() {
  local selected
  selected=$(fzf) || return
  cd -- "${selected:h}" || exit
}
```

# おまけ - エディタでコマンドラインを編集

```bash [~/.zshrc]
autoload -Uz edit-command-line
zle -N edit-command-line
bindkey '\ee' edit-command-line # <ESC>e の場合
```

# インクリメンタルサーチで最近移動した場所へ移動

```bash [~/.zshrc] {*}{maxHeight: '40%'}
export _ZO_FZF_OPTS="
  --reverse
  --style=full:rounded
  --height 75%
  --margin 0,5%
  --preview-window=down,50%,wrap
  --preview 'eza -l --icons --sort modified -r --color=always {2..}'
  --no-sort
  --exact
"
```
